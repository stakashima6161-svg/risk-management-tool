# Claude Code 作業指示書

## プロジェクト名
TKC財務リスマネ・アセスメント入門ツール（匿名化版）

## 目的
TKC会員事務所の巡回監査担当者が、関与先名や取引先名などの機密情報を入力せずに、財務諸表の数字をもとにリスクマネジメントの入口提案を行える GitHub Pages 用ツールを整備する。

このツールは、保険証券回収や保険商品の提案を目的としない。巡回監査の場で、関与先と一緒に「何が起きると困るか」「起きたらどれくらい影響があるか」「優先度は高いか」「どう備えるか」を確認するための入口資料・ヒアリング補助ツールである。

## 必須方針
- 単一HTMLで動作させる。外部API、外部CDN、外部ライブラリに依存しない。
- GitHub Pagesでそのまま公開できる形にする。
- 関与先名、住所、代表者名、担当者名、電話番号、メールアドレス、正式取引先名、正式仕入先名、証券番号は入力欄に設けない。
- 取引先は「取引先A」「取引先B」、仕入先は「仕入先X」「仕入先Y」のような匿名記号で扱う。
- AI投入用メモにも固有名詞・住所・個人情報を出さない設計にする。
- 税理士事務所の職員が使えるよう、専門用語を前面に出しすぎず、画面上は平易な表現にする。
- ただし、内部構造はリスクアセスメントの型に沿わせる。

## リスクアセスメントの流れ
1. リスクの特定
   - 主要取引先停止
   - 主要仕入先停止
   - 事業停止・収益途絶
   - 自己資本・現預金不足
   - 建物・設備停止
2. リスクの分析
   - 売上・粗利益・営業利益・現預金・固定費・借入返済への影響を計算する。
3. リスクの評価
   - 赤：早急に確認
   - 黄：次回までに確認
   - 青：現時点では大きな問題なし
4. リスク対応
   - 移転：保険、契約、外注、保証などで外部に移す
   - 回避：危険な取引や場所を避ける、取引条件を見直す
   - 保有：自己資本、現預金、緊急資金枠で自社負担する
   - 低減：分散、在庫、代替先、点検、バックアップ、BCPで影響を小さくする

## 主要機能
- 基礎財務情報入力
- 主要取引先停止リスクの簡易判定
- 主要仕入先停止リスクの簡易判定
- 事業停止・収益途絶リスクの簡易シミュレーション
- 自己資本目標の算出
- BAST等の1人当たり自己資本額を任意入力できること
- リスク対応の4分類を選択できること
- A4簡易レポート印刷
- AI投入用匿名メモ生成
- JSON保存・読込
- CSV出力

## 計算仕様
### 主要取引先停止リスク
- 停止後営業利益 = 営業利益 - 主要取引先の年間粗利益
- 粗利益依存度 = 主要取引先の年間粗利益 / 年間粗利益
- 赤：停止後営業利益がマイナス
- 黄：停止後営業利益はプラスだが、粗利益依存度が高い
- 青：影響が限定的

### 主要仕入先停止リスク
- 代替調達までの影響額 = 1日あたり対象粗利益 × 代替調達日数
- 代替仕入による追加コスト = 対象仕入額 × 原価上昇率
- 赤：代替先なし、在庫日数が短い、止まる売上が大きい
- 黄：代替先ありだが、時間・追加コストがかかる
- 青：代替先・在庫があり、影響が限定的

### 事業停止・収益途絶リスク
- 1日あたり粗利益 = 年間粗利益 / 年間営業日数
- 粗利益喪失額 = 1日あたり粗利益 × 停止日数 × 売上減少率
- 停止後営業利益 = 営業利益 - 粗利益喪失額
- 月間固定支出 = 月間固定費 + 月間借入返済額
- 無収入耐久月数 = 現預金 / 月間固定支出

### 自己資本目標
- 目標自己資本額 = BAST等の1人当たり自己資本額 × 従業員数
- 自己資本不足額 = 目標自己資本額 - 現在の自己資本額
- 現在の1人当たり自己資本額 = 現在の自己資本額 / 従業員数

## UI方針
- 入口資料なので難しくしすぎない。
- 「リスクの特定・分析・評価・対応」は画面の見出しとしては使ってよいが、説明文は平易にする。
- 巡回監査担当者が関与先に話せる30秒トークを表示する。
- 判定だけで終わらせず、次回までの確認事項を表示する。

## 禁止事項
- 実在の関与先名や取引先名を前提にしたサンプルを入れない。
- 保険商品の推奨や補償内容の断定をしない。
- 税務・法律・保険実務の確定判断をツール単体で行わない。
- 外部送信を自動で行わない。

## 今後追加したい改善候補
- 業種別テンプレート
- 製造業向け：代替生産・外注先・設備停止

- 建設業向け：元請依存・外注先停止・労災・車両事故
- 不動産業向け：家賃停止・賃借人事故・建物設備停止
- レポート文面のやさしい表現化
- 職員向け研修モード
