# Claude Code 開始プロンプト

このリポジトリは、TKC会員事務所向けの「財務リスマネ・アセスメント入門ツール」です。

まず `CLAUDE.md` と `docs/requirements.md` を読み、以下の方針で `index.html` を改善してください。

1. 単一HTML、外部APIなし、外部CDNなし、GitHub Pages対応を維持する。
2. 関与先名、住所、個人名、正式な取引先名、正式な仕入先名は入力できない設計を維持する。
3. リスクアセスメントの流れを、リスクの特定・分析・評価・対応としてわかりやすくする。
4. 入口資料なので難しくしすぎず、巡回監査担当者が関与先に説明しやすい文面にする。
5. 自己資本目標、主要取引先停止、主要仕入先停止、事業停止・収益途絶リスクを中核にする。
6. 出力レポートはA4一枚で印刷しやすくする。
7. AI投入用メモは匿名情報だけを出す。

改善後、変更点を簡潔に説明してください。
