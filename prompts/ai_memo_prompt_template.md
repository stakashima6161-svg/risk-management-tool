# AI投入用メモ生成テンプレート

以下は匿名化された財務リスマネ情報です。実在の関与先名、住所、代表者名、正式な取引先名、正式な仕入先名は含めていません。

## 依頼
巡回監査担当者が関与先に説明しやすいように、以下の観点で短く整理してください。

1. 数字から見える主なリスク
2. 最初に確認すべき事項
3. リスク対応の方向性（移転・回避・保有・低減）
4. 30秒で話せる面談トーク
5. 次回までの宿題

## 入力情報
- 業種区分：{industry_label}
- 従業員数：{employees}
- 年間売上高：{annual_sales}
- 年間粗利益：{annual_gross_profit}
- 営業利益：{operating_profit}
- 現預金：{cash_and_deposits}
- 自己資本：{equity}
- 月間固定費：{monthly_fixed_cost}
- 月間借入返済額：{monthly_debt_repayment}
- BAST等の1人当たり自己資本額：{bast_equity_per_employee}
- 主要取引先：取引先A、取引先B等の匿名記号のみ
- 主要仕入先：仕入先X、仕入先Y等の匿名記号のみ

## 注意
保険商品の推奨や補償内容の断定はしないでください。
「確認しましょう」「検討しましょう」という表現にしてください。
