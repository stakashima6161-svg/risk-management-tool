# TKC財務リスマネ・アセスメント入門ツール（匿名化版）

GitHub Pagesで動作する単一HTMLツールです。

## 運用方針

- 関与先名、正式な取引先名、住所、代表者名、担当者名、証券番号は入力しません。
- 取引先・仕入先は「取引先A」「仕入先B」などの記号で入力します。
- 金額は必要に応じて概算・百万円単位に丸めて入力します。
- GitHub Pages単体では外部APIに送信しません。
- 保存機能は利用端末のブラウザ保存です。
- AIメモを外部AIに貼り付ける場合は、各事務所の情報管理ルールに従ってください。

## 主な機能

- リスクの特定、分析、評価、対応の4ステップ
- 主要取引先停止リスク
- 主要仕入先停止リスク
- 事業停止・収益途絶リスク
- 自己資本・現預金リスク
- BAST等の1人当たり自己資本額による目標自己資本計算
- A4簡易レポート印刷
- AI投入用メモ生成
- JSON保存・読込、CSV出力
